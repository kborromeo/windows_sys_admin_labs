# System Administration Hands-on Labs
Repository of hands-on Windows system administration homelabs covering Active Directory deployment, DNS and DHCP configuration, identity and access management, and endpoint support. Designed to replicate enterprise environments using virtualization and cloud platforms, with an emphasis on troubleshooting, documentation, and real-world IT workflows.

## Hybrid Environment - Microsoft Windows / Azure
### 1. Setting up a Windows Server VM in Azure
Deploying a Windows Server 2022 VM in Microsoft Azure, configuring it as a foundation for AD DS.
